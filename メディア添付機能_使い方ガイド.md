## 🎯 概要

この機能により、以下のメディアをサジェスチョンに添付できます：

- **画像** (PNG, JPG, GIF)
- **動画** (MP4)
- **外部リンク** (Webページへのリンク)

### 表示イメージ

ユーザーがサジェスチョンをクリックすると、回答テキストと一緒にメディアが表示されます：

```
🤖 金彩は、京友禅の仕上げに金粉や金箔を使って装飾する技法じゃよ。

[📷 画像1] [📷 画像2]
[▶️ 動画: 金彩職人の技を見てみるんじゃ！]
[🔗 もっと詳しく見る →]
```

## 🚀 基本的な使い方

### 1. メディアファイルを配置

メディアファイルを`static/media/`に配置します。

```bash
# 画像を配置
static/media/kyoyuzen/新しい画像.jpg

# 動画を配置
static/media/kyoyuzen/新しい動画.mp4

# サムネイル（動画用）
static/media/thumbnails/新しい動画_thumb.jpg
```

### 2. `modules/static_qa_data.py`を編集

**重要**: CERA版の構造（質問キーワードをトップレベルキーとする）を使用します。

#### 画像を追加する場合

```python
qa_media_data = {
    "質問キーワード": {
        "images": [
            {
                "url": "/static/media/kyoyuzen/新しい画像.jpg",
                "alt": "画像の説明",
                "caption": "ユーザーに表示されるキャプション"
            }
        ]
    }
}
```

#### 動画を追加する場合

```python
qa_media_data = {
    "どうやって金を振りかけるの": {
        "videos": [
            {
                "url": "/static/media/kyoyuzen/新しい動画.mp4",
                "thumbnail": "/static/media/thumbnails/新しい動画_thumb.jpg",
                "caption": "動画の説明文",
                "alt": "動画の説明"
            }
        ]
    }
}
```

#### 外部リンクを追加する場合（NEW！）

```python
qa_media_data = {
    "京友禅について詳しく知りたい": {
        "link": {
            "text": "京友禅公式サイト →",
            "url": "https://www.kyoyuzen.example.jp/"
        }
    }
}
```

#### すべて組み合わせる場合

```python
qa_media_data = {
    "金彩について全部教えて": {
        "images": [
            {
                "url": "/static/media/kyoyuzen/example.jpg",
                "alt": "金彩の例",
                "caption": "美しい金彩じゃろ？"
            }
        ],
        "videos": [
            {
                "url": "/static/media/kyoyuzen/demo.mp4",
                "thumbnail": "/static/media/thumbnails/demo_thumb.jpg",
                "caption": "実際の作業を見るんじゃ！",
                "alt": "金彩作業"
            }
        ],
        "link": {
            "text": "もっと詳しく見る →",
            "url": "https://www.example.jp/kinsai"
        }
    }
}
```

## 📦 データ構造の詳細

### 画像オブジェクト

```python
{
    'url': str,      # 画像ファイルのパス（必須）
    'alt': str,      # 代替テキスト（アクセシビリティ用、推奨）
    'caption': str   # ユーザーに表示されるキャプション（推奨）
}
```

### 動画オブジェクト

```python
{
    'url': str,        # 動画ファイルのパス（必須）
    'thumbnail': str,  # サムネイル画像のパス（推奨）
    'caption': str,    # ユーザーに表示されるキャプション（推奨）
    'alt': str         # 代替テキスト（推奨）
}
```

### 外部リンクオブジェクト

```python
{
    'text': str,   # リンクのテキスト（必須）
    'url': str     # リンク先URL（必須）
}
```

## 🔍 質問キーワードのマッチング

### マッチングのルール

システムは以下の順序でマッチングを試みます：

1. **完全一致**: `"金彩って何"` == `"金彩って何"`
2. **正規化一致**: `"金彩って何？"` → `"金彩って何"` (疑問符を除去)
3. **部分一致**: `"金彩って何ですか"` → `"金彩って何"` (含まれていればOK)

### マッチング例

**質問キーワード**: `"金彩って何"`

以下の質問すべてにマッチします：
- ✅ 「金彩って何？」
- ✅ 「金彩って何ですか」
- ✅ 「金彩について教えて」

**質問キーワード**: `"道具は何を使うの"`

以下の質問にマッチします：
- ✅ 「道具は何を使うの？」
- ✅ 「どんな道具を使いますか」
- ✅ 「使う道具について教えて」

### 注意事項

- 大文字・小文字は区別しません
- 疑問符（？）は自動的に除去されます
- 全角・半角スペースは無視されます

## 🐛 トラブルシューティング

### メディアが表示されない

**症状**: 質問しても画像・動画が表示されない

**原因と対処法:**

1. **質問キーワードが一致していない**
   ```
   対処: 質問キーワードを見直す
   ヒント: ログに「📷 メディアヒット」が出ているか確認
   ```

2. **ファイルパスが間違っている**
   ```
   対処: ファイルパスを確認
   正: '/static/media/kyoyuzen/image.jpg'
   誤: 'static/media/kyoyuzen/image.jpg'  # 先頭の/が必要
   誤: 'kyoyuzen/image.jpg'  # /static/media/が必要
   ```

3. **ファイルが存在しない**
   ```
   対処: ファイルが実際に配置されているか確認
   コマンド: ls static/media/kyoyuzen/
   ```

4. **関数がインポートされていない**
   ```
   対処: application.pyで正しくインポートされているか確認
   ```

### 動画のサムネイルが表示されない

**対処:**
1. サムネイル画像ファイルが存在するか確認
2. サムネイルのパスが正しいか確認（`/static/media/thumbnails/...`）
3. サムネイルを省略した場合、ブラウザが自動的に最初のフレームを使用

### 外部リンクが開かない

**対処:**
1. URLが有効なHTTP/HTTPSリンクか確認
2. `url`プロパティが正しく設定されているか確認
3. ブラウザのコンソールでエラーがないか確認

## 💡 ヒント・ベストプラクティス

### 1. キャプションを工夫する

**良い例:**
```python
'caption': '金彩職人の技を見てみるんじゃ！'  # キャラクターの口調
'caption': 'これが砂子筒じゃよ！100本以上持っとるぞい'  # 詳細な説明
```

**悪い例:**
```python
'caption': '動画'  # 説明不足
'caption': 'Video file'  # キャラクター性がない
```

### 2. メディアは適度に

- すべての質問にメディアは不要
- 視覚的な説明が効果的な質問に絞る
- 動画は特に重要な技法に使用

### 3. ファイル管理

```
# おすすめのフォルダ構成
static/media/
  kyoyuzen/
    techniques/      # 技法関連
    tools/          # 道具関連
    finished/       # 完成品
    process/        # 作業工程
  thumbnails/       # すべてのサムネイル
```

### 4. ファイル命名規則

- 英数字とアンダースコアのみ使用
- 日本語は避ける
- 例: `kinsai_tools_2024.jpg`

### 5. ファイルサイズ

- 画像: 1MB以下推奨
- 動画: 10MB以下推奨（長い場合は圧縮）

## 📝 注意事項

### 1. 質問キーワードのマッチング

- サジェスチョンのテキストと`qa_media_data`のキー（質問キーワード）が一致する必要があります
- 大文字・小文字、疑問符（？）の有無は自動的に正規化されます

### 2. データ構造（CERA版準拠）

- **Phase別の階層構造は使用しません**
- 質問キーワードを直接トップレベルキーとして使用します
- 例: `qa_media_data = {"質問": {"images": [...]}}`

### 3. 多言語対応

- 日本語と英語の質問キーワードを両方登録してください
- 同じ`qa_media_data`辞書に日本語と英語のキーを並べて定義します

```python
qa_media_data = {
    # 日本語
    "金彩って何": {
        "images": [...]
    },
    # 英語
    "What is Kinsai": {
        "images": [...]
    }
}
```

### 4. メディアがない場合

- メディアを持たない質問の場合、`qa_media_data`に登録する必要はありません
- システムが自動的に`None`を返します

### 5. 外部リンクの注意事項

- `url`は有効なHTTP/HTTPSリンクを指定してください
- `text`には矢印（→）を含めると自動的にスタイリングされます
- 外部リンクは新しいタブで開かれます（`target="_blank"`）
